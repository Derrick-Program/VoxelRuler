## Account 頁面 UI 重設計（M1）

### 完成事項
- 重寫 `ui/components/pages/account.slint`：使用 `StandardTableView` 顯示帳號列表
- 更新 `ui/global.slint`：在 `PageAccountLogic` 加入 `selected-index` 與新 callbacks

### 主要設計決定
- `StandardListViewItem` 在 Slint 1.16 無法從 `std-widgets.slint` 直接 import，改用 inline struct `{ text: "..." }` 寫法
- `rows` 目前為 UI 靜態 mock data，待 M1 Rust 串接時由 Rust 動態注入
- 所有元件尺寸使用 `horizontal-stretch` / `vertical-stretch`，無固定 px 寬高

### 新增 Callbacks
`add-offline-account`, `refresh-account`, `remove-account`, `set-default-account`, `unset-default-account`, `manage-appearance`, `selected-index`

### 下一步
- Rust `view.rs` 串接 token 狀態 → 動態更新 rows
- 登出功能實作
